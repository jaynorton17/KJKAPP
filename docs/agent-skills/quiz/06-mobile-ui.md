# Mobile UI Skill

Use this when editing quiz layout, buttons, inputs, timer display, answer cards, reveal screen, or Chromebook/mobile usability.

Rules:
- Use large tap targets.
- Avoid tiny buttons and cramped panels.
- Inputs must remain easy to type into.
- Important buttons must be visible without awkward scrolling.
- Layout must work on mobile, tablet, Chromebook, and desktop.
- Avoid hover-only interactions.
- Keep question, timer, answer input/options, submit, and player status obvious.
- Reveal screen should prioritise correct answer, covered/revealed player answer boxes, green/red outcome, points gained, and Ready for next question.
